# Proyecto de Seguridad Informática

## Detección de Anomalías en Red

Este proyecto usa `Streamlit` para mostrar un dashboard interactivo de detección de anomalías.

### Librerías necesarias

- streamlit
- pandas
- numpy
- scikit-learn
- plotly

### Instalación

Ejecuta estos comandos en PowerShell dentro de la carpeta del proyecto:

```powershell
cd "c:\Users\branm\OneDrive\Documents\TercerCuatrimestre\Seguridad Informatica\Código\Detección Anomalías de red"
python -m pip install streamlit pandas numpy scikit-learn plotly
```

### Ejecución

Después de instalar las dependencias, inicia la aplicación con:

```powershell
streamlit run deteccion_anomalias_red.py
```

O si quieres usar el mismo intérprete de Python:

```powershell
python -m streamlit run deteccion_anomalias_red.py
```

### Nota

- Si Streamlit te pide un email durante el primer lanzamiento, dejar el campo vacío y presionar `Enter`.
- Asegúr de usar Python 3.10 o superior.
